lunarExpress
============
Lyfskammt
=========

###To start:###

Vera viss um að þið séuð með npm installað á tölvunni.

Clone repo:
`git clone https://github.com/gummie4444/lunarExpress.git`

Gera:
`npm install`

Gera:
`npm install --global gulp`

RUN DA GAME:
`npm gulp`

Skoðið leikinn á
`http.//localhost:8080`

To debug js code write:
`gulp jshint`
